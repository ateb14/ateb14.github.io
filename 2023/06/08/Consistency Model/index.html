Consistency Model

Posted on 2023-06-08 In Generative Models

Consistency Model

Background

Probability Flow ODE

Consider the following SDE used in diffusion model:

\[ d\bf{x_t} = \bf{\mu}(\bf{x_t},t)dt+\sigma(t)\bf{w}. \]

A probability flow ODE(PF ODE) shares the same marginal probability distribution as it:

\[ d\bf{x_t} = [\bf{\mu}(\bf{x_t},t)-\frac{1}{2}\sigma(t)^2\nabla_x \log p_t(\bf{x_t})]dt. \]

In this paper, the author proposes using \(\mu(\bf{x},t) = \bf{0}\) and \(\sigma(t) = \sqrt{2t}\) in Karras et al. (2022), which results in the empirical PF ODE:

\[ \frac{d\bf{x}_t}{d_t} = -ts_\phi(\bf{x}_t,t). \]

Definition

Given a solution trajectory solution \(\{\bf{x}_t\}_{t\in[\epsilon,T]}\) of the PF ODE, the consistency function is defined as \(\bf{f}:(\bf{x}_t,t)\mapsto\bf{x}_{\epsilon}\) which has a property called self-consistency: its outputs are consistent for arbitrary pairs of \((\bf{x}_t, t)\) that belong to the same PF ODE trajectory, i.e., \(\bf{f}(\bf{x}_t,t) = \bf{f}(\bf{x}_{t'},t')\) for all \(t,t' \in [ε,T]\).

The aforementioned definition implies a boundary condition where any \(\bf{f}(\cdot,\epsilon)\) is an identity function, meaning that \(\bf{f}(\bf{x}_\epsilon,\epsilon) = \bf{x}_\epsilon\). For consistency models based on deep neural networks, the author proposes using a skip link which enforces the model to respect the boundary condition:

\[ \bf{f}_\theta(\bf{x},t) = c_{skip}(t)\bf{x} +c_{out}F_\theta(\bf{x},t), \]

where \(c_{skip}(t)\) and \(c_{out}(t)\) are differentiable functions \(s.t.\) \(c_{skip}(\epsilon)=1\) and \(c_{out}(\epsilon)=0\).

Sampling

The sampling process involves only one forward pass through the consistency model where a sample from \(\hat{\bf{x}}_T\sim \mathcal{N}(\bf{0},T^2 I)\) is fed to the neural network \(\bf{f}_\theta(\hat{\bf{x}}_T,T)\). (Single Step)

“Importantly, one can also evaluate the consistency model multiple times by alternating denoising and noise injection steps for improved sample quality.”

Training

Consistency Distillation(CD)

Loss Function:

\[ \mathcal{L}_{CD}^N(\theta,\theta^-;\phi) := \mathbb{E}_{\bf{x} \sim p_{data}}\mathbb{E}_{n \sim \mathcal{U}[0,N-1]}[\lambda(n)d(\bf{f}_\theta(\bf{x}_{t_{n+1}},t_{n+1}),\bf{f}_{\theta^-}(\bf{\hat{x}}_{t_n}^\phi,t_{n}))], \]

where \(\bf{x}_{t_{n+1}} \sim \mathcal{N}(\bf{0},t_{n+1}^2)\) and \(\bf{\hat{x}}_{t_n}^\phi\) is estimated using a ODE solver given a pre-trained score model \(s_\phi(\bf{x},t)\):

\[ \bf{\hat{x}}_{t_n}^\phi := \bf{x}_{t_{n+1}} - (t_n-t_{n+1})t_{n+1}s_\phi(\bf{x}_{t_{n+1}},t_{n+1}). \]

\(\theta^-\) denotes the running average(EMA) of the past arguments \(\theta\) during optimization:

\[ \theta^- \leftarrow stopgrad(\mu \theta^- + 1-\mu)\theta). \]

Consistency Training(CT)

There exists an unbiased estimator of \(\nabla\log p_t(\bf{x}_t)\):

\[ \nabla\log p_t(\bf{x}_t) = -\mathbb{E}_{\bf{x}_t\sim\mathcal{N}(\bf{0},t^2I)}[\frac{\bf{x}_t-\bf{x}}{t^2}]. \]

Loss Function:

\[ \mathcal L_{CD}^N(\theta,\theta^-;\phi) = \mathbb{E}_{\bf{x} \sim p_{data}}\mathbb{E}_{n \sim \mathcal{U}[0,N-1]}[\lambda(n)d(\bf{f}_\theta(\bf{x}+t_{n+1}\bf{z},t_{n+1}),\bf{f}_{\theta^-}(\bf{x}+t_{n}\bf{z},t_{n}))], \]

where \(z\sim \mathcal N(\bf{0},I)\).

Assume that the pre-trained score model matches the ground truth \(s.t.\) \(s_\phi(\bf{x},t) \equiv \nabla \log p_t(\bf{x})\), with some additional boundary restrictions, then:

\[ \mathcal{L}_{CD}^N(\theta,\theta^-;\phi) = \mathcal{L}_{CT}^N(\theta,\theta^-;\phi) + o(\Delta t), \]

where \(\Delta t = \max_{n\in[1,N-1]}\{ |t_{n+1} - t_n|\}\).
