Sliced Score Matching

Last edited time: May 19, 2023 4:54 PM Owner: YY W Tags: Technique

Background

The optimization object of score matching is

\[ \begin{aligned} \int &p(\bold{x})[\frac{1}{2}||\nabla_x\log p_\theta (\bold{x})||^2 + tr(\nabla_x^2\log p_\theta (\bold{x}))]d\bold{x} \\&= \mathbb{E}_{x\sim p(x)}[\frac{1}{2}||\nabla_x\log p_\theta (\bold{x})||^2 + tr(\nabla_x^2\log p_\theta (\bold{x}))] \\&= \mathbb{E}_{x\sim p(x)}[\frac{1}{2}||s_\theta(x)||^2 + tr(\nabla_\bold{x}s_\theta (\bold{x}))].\end{aligned} \]

Derivation of score matching can be found on Score Matching.

However, naive score matching is NOT scalable. Suppose that we train a deep neural network which outputs a d-dimension vector as the estimation of the score function. Computing \(tr(\nabla_\bold{x}s_\theta(\bold{x}))\) requires \(O(d)\) backpropagation since we have to calculate

\[ \{ \frac{\partial s_{\theta,i}(x)}{\partial x_i}| i\in\{1,2,...,d\}\}. \]

Sliced Score Matching

Project the score function onto random vectors \(\bold{v}\), which introduces the Sliced Fisher Divergence:

\[ \frac{1}{2}\mathbb{E}_{v\sim p(v)}\mathbb{E}_{x\sim p(x)}[||\bold{v}^T\nabla_{\bold{x}}\log p_\theta(\bold{x})-\bold{v}^T\nabla_{\bold{x}}\log p(\bold{x})||^2] \\ = \mathbb{E}_{v\sim p(v)}\mathbb{E}_{x\sim p(x)}[\bold{v}^T\nabla_{\bold{x}}^2\log p_\theta(\bold{x})\bold{v}+ \frac{1}{2}[\bold{v}^T\nabla_{\bold{x}}\log p_\theta(\bold{x})]^2] + const \]

When selecting the distribution \(p(\bold{z})\), there is flexibility to choose options such as a multivariate standard normal \(N(0, I)\) or a multivariate Rademacher distribution (which is the uniform distribution over \(\{±1\}^D\)).

In practice, when working with a dataset \(X^N = \{\bold{x}_1, \bold{x}_2,...,\bold{x}_n \}\), we draw \(M\) random projections from \(p_\bold{v}\) for each point \(\bold{x}_i\) independently. The collection of all these vectors are denoted by \(V^{NM} = \{ \bold{v}_{ij}\}_{1\le i\le N, 1\le j\le M}\). The estimator can be defined as{

\[ \hat{J}(\theta;X^N,V^{NM}) = \frac{1}{N}\frac{1}{M}\sum_{i=1}^{N}\sum_{i=1}^{M}[\bold{v}_{i,j}^T\nabla_\bold{x}s(\bold{x};\theta)\bold{v}_{ij} +\frac{1}{2}\bold{v}_{ij}^Ts(\bold{x};\theta))^2] \]

When \(p_{\bold{v}}\) is a multivariate standard normal or multivariate Rademacher distribution, we have

\(\mathbb{E}_{\bold{z}\sim p(\bold{z})}[(\bold{v}_{ij}^Ts(\bold{x};\theta))^2] = ||s(\bold{x};\theta)||^2_2\), which may lead to a simplified estimator with lower variance:

\[ \hat{J}_{vr}(\theta;X^N,V^{NM}) = \frac{1}{N}\frac{1}{M}\sum_{i=1}^{N}\sum_{i=1}^{M}[\bold{v}_{i,j}^T\nabla_\bold{x}s(\bold{x};\theta)\bold{v}_{ij} +\frac{1}{2}||s(\bold{x};\theta)||_2^2] \]

The author refers to this version as sliced score matching with variance reduction(SSM-VR), and it has better performance empirically.

If we train a neural network \(h_\theta(\bold{x})\) to predict the score function, then we only need \(O(1)\) backpropagation to calculate \(\nabla_{\bold{x}}[\bold{v}^Th_{\theta}(\bold{x})] =\bold{v}^T\nabla_{\bold{x}}h_{\theta}(\bold{x})\) .

Reference

[1] Song et al(2019). Sliced Score Matching: A Scalable Approach to Density and Score Estimation.
