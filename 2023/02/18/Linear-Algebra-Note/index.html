Notes for Linear Algebra

Introduction

It’s been two years since I took my linear algebra in my freshman year, and the relevant knowledge I used in the past few semesters are so limited that I forget more than half of them. For now, I decide to read introduction to linear algebra by Gilbert Strang in order to recall the concepts of algebra and also try to think from the perspective which is somehow different from what I did in the first year.

LU factorization

We can factorize an invertible square matrix $A$ into two triangular matrices $L,U$, where $U$ is an upper triangular matrix with pivots on its diagonal, and $L$ is a lower triangular matrix with $L_{i,j}$ exactly equal to the multiplier that multiplied the pivot row $j$ when it was subtracted from row $i$ in elimination to solve $Ax = b$.
$$
A = LU
$$
$L $ stands for the forward elimination step turning $A$ into an upper triangular matrix:
$$
LUx = b \Rightarrow Ux = L^{-1}b=c \Rightarrow Lc = b
$$
$U$ stands for the backward substitution that gives the final answer after we solve $c$ From $Lc = b$:
$$
Ux = c
$$
For example, when $A = \left[ \begin{matrix} 1&2\4&9 \end{matrix} \right], b = [5,21]^{T}$, $L = \left[\begin{matrix} 1&0\4&1 \end{matrix}\right], U = \left[\begin{matrix} 1&2\0&1 \end{matrix}\right]$, for we turn $Ax=b$ into $x+2y=5,y=1$,which can also be written in the format: $Ux = c$ that yields $U$, using the forward elimination that yields $L$. In all, the whole process to solve $Ax=b$ using LU factorization can be denoted as:
$$
L \Rightarrow (c,U) \Rightarrow x
$$

The diagonal of $L$ comprises all 1’s, but $U$ probably doesn’t. We can further factorize $A$ into $LDU$ by scaling all the pivots of $U$ into one, where D is a diagonal matrix containing the pivots of the previous $U$（$A$).

It’s noteworthy that when $A^{T} = A$, $A = LDU = LDL^T$, which will save almost half of the time and space to solve $Ax=b$.

Projections

We have $n$ vectors $a_1,…,a_n$ in $R^m$ that are linearly independent and a vector $b \in R^m$. Find the vector $p=A\hat{x}$ in the column space of $A = <a_n,…,a_n>$ which is closest to $b$ and the projection matrix $P$ which statisfies $p = Pb$.

  • Subspace: $C(A)$
  • The error vector $b - A\hat{x}$ is perpendicular to this column space
  • Consequently $b-A\hat{x}$ is in the nullspace of $A^T$, which means that $A^T(b-A\hat{x}) = 0$

Therefore, $p = A\hat{x} = A(A^TA)^{-1}A^Tb \Rightarrow P = A(A^TA)^{-1}A^T$.

We have $P^2 = P$ for the second projection won’t change the first projection.

It’s noteworthy that $A^TA$ is invertible if and only if $A$ has linearly independent columns and we can prove it by proving that the nullspace of $A$ is exactly that of $A^TA$.

QR factorization

$A = QR$, where the columns of $A$ are linearly independent and $Q$ is an orthonormal matrix that comes from the Gram-Schmidt method(calculating the orthogonal bases by subtracting the projections on previous bases), i.e. $Q^TQ = I$. Therefore, $R = Q^TA$, and $q_{i}$ is orthogonal to $a_1,…,a_{i-1}$, which means that $R$ is an upper triangular matrix.

Diagonalization in Differential Equation

We try to solve this constant-coefficient differential equation starting from $\bold{u}(0)$:
$$
\frac{d\bold{u}}{dt} = A\bold{u}
$$
If $Ax = \lambda x$, we get $\frac{d\bold{u}}{dt} = \lambda \bold{u}$ when $\bold{u} = e^{\lambda t}\bold{x}$.

When $A$ is diagonalizable, the complete solution is $\bold{u(t)} = \sum_{i=1}^{i=n}c_ie^{\lambda_{i t}}\bold{x_i}$.

Even when there is a shortage of eigenvectors, we can let $\bold{u} = e^{At}\bold{u}(0)$.

  • $e^{At} = I + At + \frac{1}{2}(At)^2…$
  • $\frac{de^{At}}{dt} = Ae^{At}$
  • Eigenvalues are $e^{\lambda t}$ for $e^{At}x = (1+\lambda t+…)x$
